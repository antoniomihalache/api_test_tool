# 5. Dashboard UI

Pages implemented:
- Overview
- Services
- Authentication
- Scenarios
- Executions
- Results
- Settings

## Overview
- Recent executions table
- Success rate card
- Latency trend chart
- Running execution count
- Top-level KPI cards

## Services
- Create service with multiple environments
- View tags, namespace, environment labels
- Delete service

## Authentication
- Create auth profiles: Bearer, JWT, OAuth2, Basic, Custom
- Configure login endpoint and token extraction
- View auth flow model (VU session reuse)

## Scenarios
- No-code scenario builder (quick create)
- Scenario types: smoke/load/stress/spike/soak/custom
- Configure VUs, duration, environment
- Launch scenario directly

## Executions
- Launch tests
- Filter by status
- Monitor live status and metrics
- Cancel running execution

## Results
- Results index page for completed runs
- Detailed results page per execution
- Metric cards and latency chart
- Generate/download HTML, JSON, CSV reports

## Settings
- Monitoring links
- Runner mode notes
- API reference summary
